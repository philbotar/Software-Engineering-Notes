# Problem: 3Sum

#### Date: 7/9/25

#### Pattern Category: Two Pointers with Fixed Anchor

## My Flawed Approach: I attempted to conform every possible trio by hashing the array itself.

## The Key Insight: Anchoring a single index and only doing 2 pointers on the rest of the array.

1. Sort Array and Create return value;
2. For each value
   1. If the value is bigger than zero and this is the same as the last, skip it. (No duplicates!)
   2. Initialise the 2 pointers, low is i + 1 (so acting as mid) and high is the last value in the array).
   3. While iterating, check if larger or smaller and inc/dec accordingly.
   4. If the same add to the array, and increment j. If. J still the same as last, keep incrementing.
3. Return value;

### Optimal Code:

```
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);

        for (int i = 0; i < nums.length; i++) {
            if (i > 0 && nums[i] == nums[i-1]) {
                continue;
            }

            int j = i + 1;
            int k = nums.length - 1;

            while (j < k) {
                int total = nums[i] + nums[j] + nums[k];

                if (total > 0) {
                    k--;
                } else if (total < 0) {
                    j++;
                } else {
                    res.add(Arrays.asList(nums[i], nums[j], nums[k]));
                    j++;

                    while (nums[j] == nums[j-1] && j < k) {
                        j++;
                    }
                }
            }
        }
        return res;
    }
}
```

#### Complexity: Time: O(n^2), Space: O(1);
